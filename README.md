# Excel-Business-Performance-Dashboard
This project was created as part of my data analytics portfolio to demonstrate skills in data visualization, financial analysis, and storytelling through Excel dashboards.
# Overview
  This project presents an interactive Excel Dashboard designed to analyze business performance using key financial and operational metrics.
The dashboard provides a holistic view of revenue, expenses, profit, and transactions across multiple dimensions such as region, category, department, and customer segment.

# Dataset Details
- The dataset contains the following fields:
    - Transaction_ID – Unique identifier for each transaction
    - Transaction_Date – Date of the transaction
    - Revenue, Expenses, Profit – Financial metrics for performance tracking
    - Category, Department, Product_Line – Business dimensions for analysis
    - Region – Geographic segmentation
    - Customer_Segment – Customer classification
    - Payment_Method – Mode of payment used
    - Discount – Applied discount percentage

# Dashboard Visuals
- The dashboard includes the following key visuals:
  - Total Profit by Payment Method – Shows which payment channels are most profitable.
  - Total Revenue by Product Line – Highlights top-performing product categories.
  - Revenue / Expense / Profit by Category – Enables comparison of financial KPIs across categories.
  - Number of Transactions by Region – Provides regional distribution of business activity.
  - Average Expense for Each Category – Analyzes cost trends within each business area.

# Insights
- Identified high-revenue product lines driving majority of profit.
- Revealed payment methods contributing most to profitability.
- Highlighted cost-intensive categories requiring optimization.
- Showed regional trends and transaction concentration areas.
